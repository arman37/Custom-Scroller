Custom Scroller
==================================

The idea is to demonstrate how easily a Custom jQuery Scroller can be built.

HOW TO RUN
========
1. Clone the project from git.
2. Go to 'Custom-Scroller' directory.
3. open 'index.html' file in your browser.
